# Chat app
    Simple chat app 

## Run project in develop mode
    Run commands:
        1) npm install
        2) npm run dev

## Run project with express server
    Run commands:
        1) npm install
        2) npm run start

## Build project
    Run commands:
        1) npm install
        2) npm run buils
